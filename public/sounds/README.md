# 環境音ファイルの準備

このディレクトリには環境音ファイルを配置してください。

## 必要なファイル

### 雨シーン (rain)
- `rain-loop.mp3` - メイン: 雨音ループ
- `thunder-distant.mp3` - サブ: 遠くの雷

### 焚き火シーン (fire)
- `fire-crackle.mp3` - メイン: 焚き火の音
- `fire-pop.mp3` - サブ: 薪が爆ぜる音

### 風シーン (wind)
- `wind-loop.mp3` - メイン: 風の音
- `leaves-rustle.mp3` - サブ: 葉擦れの音

### 夜シーン (night)
- `insects-loop.mp3` - メイン: 虫の声
- `owl-distant.mp3` - サブ: 遠くのフクロウ

## 推奨フリー素材サイト

| サイト名 | URL | ライセンス |
|---------|-----|-----------|
| 効果音ラボ | https://soundeffect-lab.info/ | 商用利用可・クレジット不要 |
| DOVA-SYNDROME | https://dova-s.jp/ | 商用利用可 |
| Freesound | https://freesound.org/ | CC0素材あり |
| Mixkit | https://mixkit.co/free-sound-effects/ | ロイヤリティフリー |

## ファイル形式

- **推奨**: MP3 (128kbps以上)
- **代替**: OGG

## 配置後のテスト

1. 開発サーバーを起動: `npm run dev`
2. ブラウザでアクセス
3. 右下の音声ボタン（🔇）をクリック
4. 音声が再生されることを確認
