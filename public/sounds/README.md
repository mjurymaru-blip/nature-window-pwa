# 環境音ファイルの準備

このディレクトリには環境音ファイルを配置してください。

## シーン選択ロジック

アプリは**天気・季節・時間帯**を考慮してシーンを自動選択します。

| 優先度 | 条件 | シーン |
|-------|------|--------|
| 1 | 雨・雷雨 | 🌧 雨音 |
| 2 | 雪 | 💨 風 |
| 3 | 夏〜初秋(6-9月)の夜 | 🦗 夜（虫の声） |
| 4 | 冬・春の夜 | 💨 風 |
| 5 | 昼間(晴れ/曇り) | 💨 風（葉擦れ） |

## ファイル形式

- **推奨**: MP3 (128kbps以上)
- **代替**: OGG, WAV

## 配置後のテスト

1. 開発サーバーを起動: `npm run dev`
2. ブラウザでアクセス
3. 右下のドロップダウンでシーンを切り替えてテスト
4. コンソールで「シーンを ○○ に変更」のログを確認
